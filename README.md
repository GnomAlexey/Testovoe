==== Управление ====

Лкм - Фокусировка

ПКМ - Выбор объекта

Иконка камеры - Снять выбор со всех объектов + Возврат на стартовую позицию

ПКМ во время фокусировки - вращение вокруг объекта

Колесо мыши - Приближение/Отдаление

Fly Mode - Включение/Выключение режима полёта

Shift - ускорение 

Выделять объекты можно как через левый фрейм так и черел ПКМ

В проект можно добавить любое кол-во объектов их список генерируется автоматически
